# 🌌 Cosmic Event Tracker

A beautiful React application for tracking celestial events across the universe, built with Vite and Tailwind CSS.

## ✨ Features

- **Add Cosmic Events**: Track meteor showers, eclipses, planetary alignments, and more
- **Priority Management**: Set priority levels (Low, Medium, High, Critical)
- **Event Categories**: Organize events by type (Meteor, Eclipse, Planetary, etc.)
- **Responsive Design**: Beautiful UI that works on all devices
- **Modern Tech Stack**: Built with React 19, Vite, and Tailwind CSS

## 🚀 Getting Started

### Prerequisites

- Node.js (version 18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <your-repo-url>
cd cosmic-event-tracker
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

## 🛠️ Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production

## 📱 Features Overview

### Event Management
- Add new cosmic events with title, date, description, category, and priority
- Delete existing events
- View all events in a beautiful card layout

### Priority System
- **Critical**: Red badge for must-see events
- **High**: Orange badge for important events
- **Medium**: Yellow badge for moderate events
- **Low**: Green badge for casual events

### Responsive Design
- Mobile-first approach
- Beautiful gradient backgrounds
- Glassmorphism effects
- Smooth animations and transitions
